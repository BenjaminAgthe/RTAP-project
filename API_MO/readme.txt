This is a readme file for the SOFA API for Matlab and Octave.

Start the API: use "SOFAstart" in Matlab or Octave.
Explore the API: "demo_" files in the directory "demos".
Learn about SOFA: read the specs PDF in the directory "doc".
Report a bug: send a ticket at https://sf.net/p/sofacoustics/.
What has changed: check the file "history.txt".

Contact the main developer at piotr@majdak.com.
